## 📊 Project Summary

This project, which was realised as part of my MIT's Data Science certification, analyzes the Pima Indians Diabetes dataset to identify patterns and factors associated with diabetes risk.

The objective is to explore how medical and demographic variables (such as glucose level, BMI, age, and insulin) relate to the likelihood of diabetes, using a structured data analysis approach.

The project follows a complete data analysis workflow:
- Data exploration to understand variable distributions and data quality
- Data cleaning to handle missing or zero values in key medical features
- Exploratory Data Analysis (EDA) to identify relationships between variables and diabetes outcomes
- Visualization to highlight key trends and differences between diabetic and non-diabetic patients

## 🔍 Key Analyses

- Comparison of glucose levels between diabetic and non-diabetic individuals  
- Impact of BMI and age on diabetes risk  
- Distribution of key health indicators (insulin, blood pressure, pregnancies)  
- Detection of anomalies and unrealistic values in the dataset  

## 💡 Key Insights

- Glucose level is the strongest indicator of diabetes risk  
- Higher BMI and age are associated with a higher probability of diabetes  
- The dataset contains several zero or missing values that require careful preprocessing  
- Clear differences exist between diabetic and non-diabetic groups across multiple variables  

This project demonstrates the ability to clean medical data, perform exploratory analysis, and extract meaningful insights to support healthcare-related decision-making.
