## 📊 Résumé du projet

Ce projet réalisé dans le cadre d'une certification du MIT-Schwarzman Computing College, analyse le dataset Pima Indians Diabetes afin d’identifier les facteurs associés au risque de diabète.

L’objectif est d’étudier comment différentes variables médicales et démographiques (glucose, IMC, âge, insuline, etc.) influencent la probabilité d’être diabétique, à travers une approche structurée d’analyse de données.

Le projet suit un workflow complet :
- Exploration des données pour comprendre leur structure et leur qualité  
- Nettoyage des données (gestion des valeurs nulles ou incohérentes, notamment les zéros)  
- Analyse exploratoire (EDA) pour identifier les relations entre variables et diabète  
- Visualisation pour mettre en évidence les différences entre individus diabétiques et non diabétiques  

## 🔍 Analyses réalisées

- Comparaison des niveaux de glucose entre patients diabétiques et non diabétiques  
- Impact de l’IMC et de l’âge sur le risque de diabète  
- Distribution des variables clés (insuline, pression sanguine, grossesses)  
- Détection de valeurs aberrantes ou incohérentes  

## 💡 Résultats clés

- Le niveau de glucose est le facteur le plus déterminant  
- Un IMC et un âge élevés augmentent la probabilité de diabète  
- Le dataset contient des valeurs manquantes déguisées en zéros  
- Des différences nettes apparaissent entre les deux groupes (diabétiques vs non diabétiques)
